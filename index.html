Phonebook app

Description

You should build your contact list, in the beginning, the list will be empty, but you can add, remove or block one contact by dispatching redux action

contact structure

every contact should have (at least)

  • name
  • email
  • phone number

Acceptance criteria (AC)

add a contact delete a contact block a contact log currently non-blocked list of contacts

  • Add a contact
  • Delete a contact
  • Block a contact
  • Log currently non-blocked list of contacts

  • Delete a contact
  • Block a contact

Application API

  • addContact(contact) Add contact
  • deleteContact(id) Delete contact
  • blockContact(id) Block contact
  • getWhiteList() Get all unblocked contacts
  • getBlackList() Get all blocked contacts
  • getAll() Get all contacts
